 CLI-приложение на Python, PostgreSQL и SQLAlchemy ORM.
Приложение  ведет складской учёт: добавляет товары, категории, поставщиков, складские операции и смотреть отчёты по остаткам.

создать докер с данными: docker run --name inventory_manager cki -e POSTGRES_DB=invantory_manager_db -e POSTGRES_USER=inventory_user -p 5435:5432 -v inventory_manager_data:/var/lib/posrtgresql/data -d postgres:16 

cоздать соединение dbeaver

в терминале ввести:  python create_tables.py ; python seed.py ;  python main.py                                

должно выйти меню:
1. Создать категорию
 2. Показать все категории
 3. Создать поставщика
 4. Показать всех поставщиков
 5. Деактивировать поставщика
 6. Создать товар
 7. Показать все товары
 8. Показать товар по id
 9. Обновить цену товара
10. Деактивировать товар
11. Добавить поступление товара
12. Добавить списание товара
13. Добавить корректировку остатка
14. Показать историю операций
15. Показать операции по товару
16. Показать текущие остатки
17. Показать товары, которые заканчиваются
18. Показать общую стоимость склада
19. Показать товары по категориям
20. Показать товары по поставщикам
 0. Выход
